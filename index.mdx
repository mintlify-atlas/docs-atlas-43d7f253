## Built With

This project leverages modern technologies for high performance and developer experience:

- **Bun** - Fast JavaScript runtime with built-in TypeScript support
- **Hono** - Ultrafast web framework optimized for edge computing
- **Redis** - In-memory data store for job queue management
- **BullMQ** - Premium message queue for Node.js based on Redis
- **WhatsApp Web.js** - WhatsApp Web API client for Node.js
- **Prisma** - Next-generation ORM for type-safe database access
- **SQLite** - Lightweight embedded database for data persistence
