---
title: 'List All Messages'
api: 'GET /messages/all'
---

Retrieve all scheduled messages with recipient details, ordered by send time.

## Authentication

This endpoint requires Basic Authentication.

<ParamField header="Authorization" type="string" required>
  Basic Auth credentials in the format `Basic base64(user:pass)`
</ParamField>

## Response

<ResponseField name="messages" type="array">
  Array of message objects
  
  <ResponseField name="id" type="string">
    Unique message identifier
  </ResponseField>
  
  <ResponseField name="content" type="string">
    Message content to be sent
  </ResponseField>
  
  <ResponseField name="sendToPhone" type="string">
    Recipient's phone number
  </ResponseField>
  
  <ResponseField name="sendAfter" type="number">
    Number of days after which to send the message
  </ResponseField>
  
  <ResponseField name="createdAt" type="string">
    ISO 8601 timestamp when message was created
  </ResponseField>
  
  <ResponseField name="sendTo" type="object">
    Recipient details
    
    <ResponseField name="name" type="string">
      Recipient's name
    </ResponseField>
  </ResponseField>
</ResponseField>

## Example Request

```bash
curl -X GET http://localhost:3000/messages/all \
  -H "Authorization: Basic $(echo -n 'user:pass' | base64)"
```
